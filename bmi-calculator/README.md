# BMI計算アプリ

身長と体重からBMIを計算する、シンプルなPHPアプリです。

入力された値をPHPで受け取り、BMIの計算結果と判定を表示します。

## 主な機能

* 身長と体重の入力
* POSTによるフォームデータの送信
* BMIの計算
* 小数第1位までの結果表示
* BMIに応じた判定表示
* 未入力や不正な値のチェック
* 入力した値の再表示

## BMIの判定

* 18.5未満：やせ気味
* 18.5以上25未満：標準範囲
* 25以上：太り気味

## 実行方法

このアプリはPHPを使用しているため、GitHub Pagesでは実行できません。

XAMPPなどのPHP実行環境が必要です。

### 1. プロジェクトを配置する

XAMPPの`htdocs`フォルダ内にプロジェクトを配置します。


C:\xampp\htdocs\bmi-calculator


### 2. Apacheを起動する

XAMPP Control PanelからApacheを起動します。

### 3. ブラウザで開く


http://localhost/bmi-calculator/


## 処理の流れ

1. 身長と体重を入力する
2. フォームからPOSTでデータを送信する
3. PHPが入力値を確認する
4. 身長をメートルに変換する
5. BMIを計算して小数第1位に丸める
6. BMIの値に応じて判定を表示する

## 入力チェック

以下の場合はエラーメッセージを表示します。

* 身長または体重が未入力
* 数値以外が入力された場合
* 0以下の値が入力された場合
