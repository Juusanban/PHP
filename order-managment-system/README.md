# 注文管理システム

PHPとSQLiteを使用して作成した、シンプルな注文管理システムです。

商品名、価格、注文者を入力して注文を追加できます。
登録した注文は一覧画面に表示され、価格の変更や注文の削除もできます。

データベースとの接続にはPDOを使用し、データベースに関する処理は`OrderDB`クラスにまとめています。

## 主な機能

* 注文の追加
* 注文一覧の表示
* 注文価格の変更
* 注文の削除
* 注文日時の自動保存
* 入力内容のチェック
* SQLiteへのデータ保存
* PDOによるデータベース接続
* Prepared StatementによるSQL実行

## CRUD処理

このアプリでは、基本的なCRUD処理を実装しています。

* Create：注文の追加
* Read：注文一覧の表示
* Update：注文価格の変更
* Delete：注文の削除

## 各ファイルの説明

### index.php

登録されている注文を一覧表示します。

このページから注文の追加画面と価格変更画面へ移動できます。
注文の削除もこのページで行います。

### add_order.php

新しい注文を追加するための画面です。

以下の情報を入力します。

* 商品名
* 価格
* 注文者

未入力や不正な価格がある場合は、エラーメッセージを表示します。

### update_order.php

登録済みの注文を選択して、価格を変更するための画面です。

注文はプルダウンメニューから選択できます。

### db.php

SQLiteへの接続とデータベース操作を行う`OrderDB`クラスを定義しています。

以下のメソッドがあります。

* `insertOrder()`：注文を追加する
* `getAllOrders()`：すべての注文を取得する
* `getOrderById()`：指定したIDの注文を取得する
* `updatePrice()`：注文価格を変更する
* `deleteOrder()`：注文を削除する

また、必要なテーブルが存在しない場合は自動で作成します。

### style.css

注文一覧、入力フォーム、ボタンなどのデザインを設定します。

## データベース

SQLiteの`order.db`を使用しています。

使用するテーブルは`T_ORDER`です。

主なカラムは以下のとおりです。

* `ID`：注文ID
* `ITEM_NAME`：商品名
* `PRICE`：価格
* `CUSTOMER`：注文者
* `ORDER_DATE`：注文日時

## 入力チェック

注文追加画面では、以下の内容を確認します。

* すべての項目が入力されているか
* 価格が数値か
* 価格が1円以上か

価格変更画面では、以下の内容を確認します。

* 注文が選択されているか
* 新しい価格が入力されているか
* 価格が数値か
* 価格が1円以上か
* 指定された注文が存在するか

## データベース処理

SQLの実行にはPDOの`prepare()`と`execute()`を使用しています。

ユーザーが入力した値をSQL文に直接連結せず、プレースホルダーを使ってデータを渡しています。

データベース処理は`OrderDB`クラスにまとめ、画面処理と分けています。

## 画面表示の安全対策

データベースから取得した商品名や注文者名を画面に表示する際は、`htmlspecialchars()`を使用しています。

追加、変更、削除の処理後は`index.php`へリダイレクトし、ブラウザを更新したときに同じ処理が繰り返されないようにしています。

## 実行方法

このアプリはPHPとSQLiteを使用しているため、GitHub Pagesでは実行できません。

XAMPPなどのPHP実行環境が必要です。

### 1. プロジェクトを配置する

XAMPPの`htdocs`フォルダ内にプロジェクトを配置します。


C:\xampp\htdocs\order-management-system


### 2. Apacheを起動する

XAMPP Control PanelからApacheを起動します。

### 3. ブラウザで開く


http://localhost/order-management-system/


現在のローカル環境で別のフォルダ内に配置している場合は、そのフォルダ構成に合わせてURLを変更してください。

## 処理の流れ

### 注文追加

1. 商品名、価格、注文者を入力する
2. PHPが入力内容を確認する
3. `OrderDB`クラスの`insertOrder()`を実行する
4. 注文日時と一緒にSQLiteへ保存する
5. 注文一覧画面へ戻る

### 価格変更

1. 登録済みの注文を選択する
2. 新しい価格を入力する
3. 対象の注文が存在するか確認する
4. `updatePrice()`を実行する
5. 注文一覧画面へ戻る

### 注文削除

1. 注文一覧の削除ボタンを押す
2. POSTで注文IDを送信する
3. 対象の注文が存在するか確認する
4. `deleteOrder()`を実行する
5. 注文一覧画面を再表示する
