# ファイル管理アプリ

PHPを使用して作成した、シンプルなファイル管理アプリです。

ファイルをサーバーにアップロードし、アップロード済みファイルの一覧からダウンロードできます。

## 主な機能

* ファイルのアップロード
* アップロード済みファイルの一覧表示
* ファイルのダウンロード
* 最大ファイルサイズのチェック
* 同じ名前のファイルの重複チェック
* アップロード結果のメッセージ表示
* `uploads`フォルダの自動作成

## 使用技術

* PHP
* HTML
* CSS
* HTTPファイルアップロード

## ファイル構成


simple-file-manager/
├── uploads/
├── index.php
├── download.php
├── style.css
└── README.md

## 各ファイルの説明

### `index.php`

ファイルのアップロード処理、入力チェック、ファイル一覧の表示を行います。

### `download.php`

指定されたファイルを`uploads`フォルダから取得し、ダウンロード処理を行います。

### `style.css`

画面のデザインを設定します。

### `uploads`

アップロードされたファイルを保存するフォルダです。

## アップロードの条件

最大ファイルサイズは4MBです。

同じ名前のファイルがすでに存在する場合は、アップロードできません。

## 実行方法

このアプリはPHPを使用しているため、GitHub Pagesでは実行できません。

XAMPPなどのPHP実行環境が必要です。

### 1. プロジェクトを配置する

XAMPPの`htdocs`フォルダ内にプロジェクトを配置します。


C:\xampp\htdocs\simple-file-manager


### 2. Apacheを起動する

XAMPP Control PanelからApacheを起動します。

### 3. ブラウザで開く


http://localhost/simple-file-manager/


## 処理の流れ

1. ユーザーがファイルを選択する
2. フォームからファイルをPOSTで送信する
3. PHPがアップロードエラーとファイルサイズを確認する
4. 同じ名前のファイルが存在しないか確認する
5. ファイルを`uploads`フォルダに保存する
6. 保存済みファイルを一覧表示する
7. ダウンロードボタンからファイルを取得する

## セキュリティ対策

ダウンロード時は`basename()`を使用し、フォルダ外のファイルが指定されないようにしています。

画面にファイル名を表示する際は`htmlspecialchars()`を使用しています。
